Allows to record GPS, NMEA, battery level, BLE heart rate and all the phone sensors data. They can be saved to a text file, a binary file or directly over a network using UDP or TCP protocols.
A collection of Python scripts for processing recorded files can be found on https://github.com/mrwojtek/press-alt project page. They were primarily written to improve the altitude estimate by combining GPS and pressure measurements.
This app is an Open Source software published on https://github.com/mrwojtek/sens-rec under Apache License 2.0.
